# House Price Prediction using Linear Regression

## Project Overview

This project demonstrates how Linear Regression can be used to predict house prices based on house area. The goal is to understand the fundamentals of supervised machine learning and regression analysis.

## Dataset

The dataset contains:

* Area (square feet)
* House Price (lakhs)

## Technologies Used

* Python
* Pandas
* NumPy
* Scikit-learn
* Matplotlib

## Steps Performed

1. Loaded the dataset using Pandas.
2. Selected Area as the feature (X).
3. Selected Price as the target variable (y).
4. Trained a Linear Regression model.
5. Generated predictions for new house sizes.
6. Evaluated the model using R² Score.
7. Visualized the regression line and data points.

## Model Equation

Price = Intercept + (Coefficient × Area)

## Results

The model achieved an R² score of approximately 0.997, indicating a strong linear relationship between house area and price.

## Learning Outcomes

* Understanding Linear Regression
* Feature and target selection
* Model training and prediction
* Model evaluation using R² Score
* Data visualization
